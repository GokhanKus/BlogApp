# BlogApp MVC

### Some operations I used for this project
login, logout, register, tags, ajax, libman, viewcomponent, one-to-many & many-to-many relations between table


## Used Technologies
- **Mvc Design Pattern**
- **N-tier Architecture**
- **EF Core ORM**
- **SQLite DataBase**
- **MySql**
- **BootStrap**
- **Tinymce Html Editor**
